# Maze Solving Robot

An autonomous Maze Solving Robot developed using **Arduino Nano**, **HC-SR04 Ultrasonic Sensors**, **L298N Motor Driver**, and **DC BO Motors**. The robot detects walls and obstacles, makes real-time navigation decisions, and finds the correct path through a maze without human intervention. 

# Project Overview

The Maze Solving Robot is an embedded robotics project that demonstrates autonomous navigation using ultrasonic sensors and a microcontroller. The robot continuously senses its surroundings, detects obstacles, and navigates through a maze using a logic-based pathfinding algorithm.

This project helps in understanding robotics, embedded systems, automation, and real-time decision making.

# Working Principle

1. Ultrasonic sensors continuously detect obstacles.
2. Distance data is sent to the Arduino Nano.
3. The Arduino processes sensor readings.
4. The programmed navigation algorithm decides the movement.
5. The L298N Motor Driver controls the DC motors.
6. The robot moves forward or turns left/right based on sensor data until it reaches the destination. 



# Maze Solving Algorithm

The robot uses a logic-based maze-solving technique such as:

- Left-Hand Rule
- Right-Hand Rule

The algorithm enables the robot to detect paths, avoid dead ends, and reach the destination autonomously.


# How to Run

1. Connect all hardware components.
2. Open the Arduino code in Arduino IDE.
3. Select the correct Arduino Nano board.
4. Select the COM Port.
5. Upload the program.
6. Place the robot at the maze entrance.
7. Power ON the robot.
8. The robot will autonomously solve the maze.
